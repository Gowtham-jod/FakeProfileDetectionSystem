# Fake Profile Detection System

## 📌 Overview
This project uses Machine Learning to detect fake social media profiles based on user activity.

## 🚀 Features
- Detect fake vs real profiles
- Random Forest ML model
- Streamlit web interface

## 🛠 Tech Stack
- Python
- Scikit-learn
- Pandas
- Streamlit

## ▶️ How to Run

1. Install dependencies:
   pip install pandas numpy scikit-learn streamlit

2. Train model:
   python train.py

3. Run app:
   streamlit run app.py

## 📊 Dataset
Use a dataset with features like:
followers, following, posts, bio_length, etc.

## 📌 Output
- Fake Profile 🚨
- Genuine Profile ✅
